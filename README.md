# Mars Web Scraping

My objecteive was to scrape the various websites listed below. Compile the data into a database, and use that to build website of my own. Using Python, Splinter, MongoDB, Beautiful Soup, HTMl, and Pandas.

- https://astrogeology.usgs.gov/
- https://space-facts.com/mars/
- https://twitter.com/MarsWxReport
- https://www.jpl.nasa.gov
- https://mars.nasa.gov/news/'
